------------------------------------
--Instalando Python+Django Windows--
------------------------------------
01. Instalar python 2.7
02. Executar comando: python -m pip install -U pip
03. Executar comando: pip install virtualenv
04. Dentro da pasta do projeto executar o comando:
python -m virtualenv helpdesk_nortec
05. Download do git: https://git-scm.com/download/win
06. Instalar git, com opção console.
07. Scripts\activate
08. Entrar no bitbucket copiar link em clone, e executar comando
09. Abrir repositorio e executar o comando: git status
10. git checkout master
11. Baixar atualizações: git pull origin master
12. instalar pacotes: pip install -r requirements.txt 
13. Baixar o mysql-connector-c e instalar
14. Baixar MySQL-python-1.2.5 e instalar
15. Baixar Microsoft Visual C++ Compiler for Python 2.7 e instalar
16. Utilizar comando "easy_install MySQL-python" para instal o mysql-python.
17. Criar database.
18. python manage.py migrate
19. python manage.py createsuperuser

Obs: Caso não tenha configurado o path, os comandos não
serão executados. Exemplo do comando a ser utilizado: 
c:\python27\python -m pip install -U pip
